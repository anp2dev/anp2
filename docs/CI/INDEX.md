## Why this file exists

A protocol's claim that it "incorporates community input" is cheap to
make and hard to verify. This index makes it verifiable: every
substantive critique we receive in public channels appears here with
permalink, contributor identifier, and outcome. A first-time visitor
should be able to confirm in 60 seconds that we actually do what we
say we do.
